About the project

What is Oh My Grill?

Oh My Grill! is an interactive cooking game where players take on the role of chefs working together to fulfill orders as quickly as possible. Each player is responsible for different tasks, such as grilling, assembling, or delivering dishes, creating a fast-paced and collaborative experience.

The project focuses on real-time multiplayer interaction using Multipeer Connectivity, allowing nearby players to seamlessly connect and play together. Built with SpriteKit, the game emphasizes responsive controls, smooth animations, and a playful visual style, while CloudKit is used to support data persistence and player progression.

WanderFeel.

About the project

What is Wanderfeel?

WanderFeel is an arcade-style mobile game focused on quick reflexes and simple mechanics. The player controls a sardine jumping between platforms, trying to survive as long as possible while the difficulty gradually increases.

The game was built using SpriteKit, with an emphasis on smooth physics, responsive controls, and a playful visual identity.
